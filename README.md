# AI Chatbot (Python)

This project is a simple AI chatbot built using Python and OpenAI API.

## Features
- Chat-based interaction
- Uses LLM (GPT model)
- Real-time response generation

## Tech Stack
- Python
- OpenAI API

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your API key

3. Run:
   python app.py

## Example
You: Hello  
Bot: Hi! How can I help you?
